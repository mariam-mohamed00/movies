# 🎬 Movie App - Flutter

A feature-rich Flutter application for discovering movies, exploring categories, and managing a personal watchlist. This app leverages the TMDB API for real-time movie data and Firebase Firestore for persistent watchlist management.

---

## ✨ Features

- **🏠 Home Screen**:
  - **Popular Movies**: Explore a curated list of currently popular movies.
  - **New Releases**: Stay updated with the latest and upcoming films.
  - **Recommended**: Discover top-rated movies recommended just for you.
- **🔍 Search**: Find any movie by title with a real-time search interface.
- **📂 Browse**: Navigate movies by specific genres and categories.
- **📌 Watchlist**:
  - Add movies to your personal watchlist.
  - View and manage your saved movies.
  - Persistent storage using Firebase Firestore.
- **ℹ️ Movie Details**: 
  - Comprehensive details for each movie (Synopsis, release date, rating).
  - List of similar movies to keep exploring.
- **🎨 Modern UI/UX**:
  - Dark-themed design for a premium cinematic feel.
  - Smooth transitions and interactive carousels.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev/) (Dart)
- **State Management**: standard `setState` (extensible to Provider/BLoC)
- **Backend/Database**: [Firebase Firestore](https://firebase.google.com/products/firestore)
- **API**: [TMDB (The Movie Database) API](https://www.themoviedb.org/documentation/api)
- **Libraries & Plugins**:
  - `http`: For network requests.
  - `carousel_slider`: For dynamic movie posters.
  - `cached_network_image`: For efficient image loading and caching.
  - `easy_search_bar`: For a seamless search experience.
  - `firebase_core`: For Firebase initialization.
